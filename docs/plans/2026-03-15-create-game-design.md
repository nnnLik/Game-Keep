# Create Game — Design

## Цель

Позволить пользователю добавлять игру в коллекцию через FAB и модальное окно. Форма расширяема для будущих полей (картинка, Steam, рейтинг).

## Решения

### UI

- **FAB** — фиксированная кнопка в правом нижнем углу, иконка plus. Видна только на `/`.
- **Модалка** — overlay с затемнённым фоном. Клик по фону / Escape / «Отмена» закрывает.
- **Форма** — name (текст), state (dropdown), is_favorite (чекбокс). Секция «Дополнительно» (collapsed) — заглушка для будущих полей.

### Черновик

- Ключ localStorage: `gametrack_create_game_draft`
- Сохранять при закрытии без submit (overlay, «Отмена», Escape)
- Очищать при успешной отправке
- При открытии — подставлять данные из черновика

### Валидация

- name: обязательное, min 1 символ
- state: обязательное, один из backlog | in_progress | completed | abandoned

### Расширяемость

- Секция «Дополнительно» — collapsed по умолчанию. В будущем: image_url, steam_url, rating и т.д.
