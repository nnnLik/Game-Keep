# Редактирование и удаление карточки игры — дизайн

## Цель

Добавить возможность редактировать все поля карточки игры и удалять её. Редактирование — через модалку, удаление — с подтверждением на странице игры.

## Контекст

- **Редактирование:** все поля как в CreateGameModal (название, обложка, Steam URL, жанры, стейт, заметка, даты, часы).
- **Удаление:** кнопка на странице игры с диалогом подтверждения (UI kit).
- **Сердечко (избранное):** уже реализовано — PATCH для is_favorite, отображение в списках.

## Архитектура

### Backend

- **PATCH /users/me/games/:id** — расширить: принимать все поля карточки (сейчас только is_favorite). Валидация как в CreateGame.
- **DELETE /users/me/games/:id** — новый эндпоинт. 204 No Content. Проверка владельца.
- Сервисы: `UpdateGameService` (или расширить `UpdateGameFavoriteService`), `DeleteGameService`.
- DAO: `update()`, `delete()` в UserGameDAO.

### Frontend

- **GameFormFields.vue** — общий компонент полей формы (картинка, название, Steam URL, жанры, стейт, заметка, даты, часы).
- **CreateGameModal** — рефакторинг: использовать GameFormFields на шаге 2.
- **EditGameModal** — модалка с GameFormFields, кнопки «Сохранить» / «Отмена».
- **Страница игры** — кнопки «Редактировать» и «Удалить» (только владельцу). Удаление — UModal/UAlertDialog.
- **Index** — иконка редактирования в карточке игры (только свои игры).

## Поток данных

### Редактирование

1. Владелец нажимает «Редактировать» → открывается EditGameModal с данными игры.
2. Меняет поля → «Сохранить» → PATCH → закрытие модалки, обновление game на странице.
3. На index: клик по иконке редактирования → переход на `/games/:id` и открытие модалки (или сразу модалка, если доступна).

### Удаление

1. Владелец нажимает «Удалить» → UModal/UAlertDialog «Удалить игру? Действие необратимо».
2. Подтверждение → DELETE → редирект на `/`.

## Размещение кнопок

- **Страница игры:** в hero-блоке рядом со статусом — иконка карандаша (редактировать), иконка корзины (удалить).
- **Index:** в карточке игры — иконка карандаша справа (рядом с сердечком).

## Ошибки

Toast при ошибках API (валидация, 404, сеть).
