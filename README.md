## Game&Keep — персональный трекер игр

### Запуск

**Бэкенд** (FastAPI):
```bash
cd backend/src && uv run python main.py
```

**Фронтенд** (Nuxt 4):
```bash
cd frontend && pnpm install && pnpm dev
```

Фронт: http://localhost:3000 | API: http://localhost:9999

### Концепция

Game&Keep — это минималистичный веб-сервис для тех, кто хочет осмысленно вести свою игровую историю. Никакого сообщества, никаких друзей, никаких рекомендаций на старте. Только ты и твои игры.

Проект родился из личной потребности: хранить впечатления об играх в структурированном виде, видеть прогресс и не забывать, во что хотел поиграть. Существующие решения либо перегружены социальщиной, либо требуют ручного заполнения тонны полей.

### Функциональность MVP

**Авторизация**
Простая JWT-авторизация по email/паролю. Никаких сторонних сервисов. Только ты имеешь доступ к своей коллекции.

**Профиль пользователя**
- Аватарка (загружается или гравикон по умолчанию)
- Юзернейм
- Короткое описание/статус (опционально)

**Навигация**
Две вкладки в левом сайдбаре: «Профиль» и «Лента». Весь интерфейс отцентрирован по центру экрана, большие воздушные поля по бокам. Никаких лишних элементов. Тёмная тема по умолчанию.

**Вкладка «Профиль» — управление коллекцией**
Под шапкой профиля расположены пять горизонтальных табов:

- `Бэклог` — игры, в которые хочется сыграть
- `В процессе` — то, во что играешь прямо сейчас
- `Пройдено` — игры, которые довел до конца
- `Брошено` — те, что не зашли
- `Любимое` — избранное, отдельный таб для быстрого доступа

Внутри каждого таба — сетка карточек игр. Карточка содержит:
- Обложка (крупно)
- Название игры
- Твоя оценка (если есть) — просто цифра в углу карточки

**Создание и редактирование игр**
При нажатии на кнопку «Добавить игру» открывается минимальная форма:
- Поле для ссылки на Steam (если введена — подтягиваем название, обложку, жанр, год, разработчика автоматически через Steam API)
- Если ссылки нет — ручной ввод: название и год (остальное заполняется позже)
- Выбор статуса при создании (по умолчанию «Бэклог»)

При клике на карточку открывается детальный вид:
- Вся информация об игре (обложка, название, год, жанр, разработчик)
- Блок с критериями оценки (пользователь сам настраивает свой набор критериев в настройках)
- Текстовое поле для заметок
- Кнопка смены статуса
- Возможность удалить игру из коллекции

Оценка проставляется по каждому критерию (ползунки или цифры). Итоговая оценка считается как среднее. Оценку можно поставить на любом этапе, даже для игр в «Бэклоге» (например, если уже играл раньше, но на другом носителе).

**Вкладка «Лента»**
Хронологическая лента всех твоих действий:
- Добавил игру (с ссылкой на карточку)
- Переместил игру из статуса в статус
- Поставил оценку
- Написал заметку

Каждое действие кликабельно — ведет на карточку игры. Позже, когда появятся другие пользователи, лента сможет показывать и их активность, но на MVP только свои действия.

**Настройки (вынесены в меню профиля)**
- Редактирование профиля (аватар, юзернейм, описание)
- Настройка критериев оценки: можно создавать свои критерии с любыми названиями, удалять, менять порядок. Никаких предустановленных списков.
